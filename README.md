# Wage Converter

Wage Converter is a browser extension that helps you visualise the true cost of items and products on e-commerce webistes like Amazon, Etsy, Ebay, etc. 

## Feautres

* **Real-time Conversion**: Automatically calculates and displays the work hours you need to work in order to afford a specifc product.
* **Multi-Currency Support**: Set your wage in your preferred currency (USD, EUR, GBP, INR, JPY) and convert effortlessly.
* **Simple User Interface**: The extension offers a simple and no-nonsense UI.
* **Universal Applicability**: Works on almost all e-commerce websites that list out prices in the supported currencies.

## How to Install

1. Download or clone this repo to your computer.
2. Open Google Chrome and navigate to `chrome://extenions/`.
3. Toggle **Developer Mode** in the top right corner.
4. Click **Load unpacked**.
5. Select the folder containing these files.

## How to Use

1. Click the extension icon in your toolbar.
2. Select your base currency and enter your hourly wage, then click **Save Wage**.
3. Now when you visit any e-commerce platform, you should see the hours you need to work to earn that specific product.


## Built With 

* **JavaScript**: Logic for DOM manipulation and calculation.
* **Chrome Extension API**: For storage, tab interaction, and scripting.
